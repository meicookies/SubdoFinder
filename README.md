# Cara Pakai
Buka terminal, lalu copy text yg ada di bawah ini
* ``` sudo apt install git curl jq sed -y ```
* ``` git clone https://github.com/mugi789/SubdoFinder/ ```
* ``` cd /SubdoFinder/ ```
* ``` chmod +x subfind.sh ```
* ``` ./subfind.sh ```

**Tested on Ubuntu 18.04**
